# CSV folder

This folder contains all csv files used and produced when running the code. Subfolders have their own readme files that explain what they are for. But their purpose is also listed here.

- dataset: final datasets for training env+nifh data together
- features: individual features after some processing, for instance only nifh table
- filled: features after imputing the values where necessary and extrapolation
- nifh: raw nifh data from the source
- nifh_di: raw nifh data depth integrated
- predictions: final predictions based on the data.
